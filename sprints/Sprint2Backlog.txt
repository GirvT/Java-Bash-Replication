User Stories from Product Backlog that we will address in this sprint:

a) As a user, I should be able to exit the program
b) As a user, I should be able to create a new directory with a specified name relative to the current directory 
c) As a user, I should be able to create a new directory with a specified name with a full path
d) As a user, I should be able to change their current directory to a new directory relative to the current directory
e) As a user, I should be able to change their current directory to a new directory using a full path
f) As a user, I should be able to print the contents of their current directory
g) As a user, I should be able to print the contents of the specified file
h) As a user, I should be able to print the contents of the specified directory
i) As a user, I should be able to print the PATH to the current working directory
j) As a user, I should be able to store working directories in the form of a stack
k) As a user, I should be able to retrieve working directories from the stack
l) As a user, I should be able to set the specified directory as new working directory by pushing the old working directory into the stack
m) As a user, I should be able to retrieve an old working directory from the stack and set that as the new working directory
n) As a user, I should be able to view a number of commands used in the terminal, relative to most recent use
o) As a user, I should be able to view a file printout in the terminal
p) As a user, I should be able to input a message and get the same message back
q) As a user, I should be able to replace text in a text file
r) As a user, I should be able to append text to a text file
